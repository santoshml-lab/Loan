## 📌 Problem Statement
Financial institutions need fast, reliable systems to assess loan risk.  
Manual evaluation is slow and inconsistent.

This project solves that by:
- Predicting default probability  
- Automating risk-based decision making  
- Providing explainable insights  

---

## 🧠 Key Features

- 📊 **Risk Prediction Model** (Default / No Default)
- 🎯 **Decision Engine**
  - Approve ✅
  - Manual Review ⚠️
  - Reject ❌
- 📈 **Risk Score Visualization** (Progress Bar / Meter)
- 🧠 **Explainable AI (XAI)**
  - Shows why a decision was made
- 💻 **Interactive UI**
  - Built using Streamlit

---

## ⚙️ Tech Stack

- Python  
- scikit-learn  
- Streamlit  
- NumPy / Pandas  

---

## 🏗️ Project Architecture
User Input → ML Model → Risk Score → Decision Engine → UI Output

---

## 📊 Model Details

- Algorithm: Random Forest Classifier  
- Task: Binary Classification (Default Prediction)  
- Evaluation Metric: Accuracy (~88%)  

---

## 🧾 Decision Logic

| Risk Score | Decision |
|------------|----------|
| < 30%      | Approve |
| 30% – 70%  | Manual Review |
| > 70%      | Reject |

---

## 🧠 Explainability

The system provides reasoning such as:
- High loan-to-income ratio  
- Unemployment risk  
- Financial stability indicators
